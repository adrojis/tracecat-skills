# CLAUDE.md

This file provides guidance to Claude Code when working with this repository.

## Project Overview

This is the **tracecat-skills** repository - a collection of Claude Code skills for building Tracecat SOAR workflows using the tracecat-mcp MCP server.

**Repository**: https://github.com/adrojis/tracecat-skills

## The 5 Skills

### 1. tracecat-action-configuration
Configure Tracecat workflow actions correctly (HTTP requests, transforms, AI actions, Python scripts, loops, conditions, retry policies).

### 2. tracecat-case-management
Create and manage Tracecat cases, incidents, status updates, comments, custom fields, and case automation.

### 3. tracecat-mcp-tools-expert
Expert guide for using Tracecat MCP tools and API effectively (workflows, cases, executions, schedules).

### 4. tracecat-secrets-integrations
Configure integrations and secrets for Splunk, CrowdStrike, Okta, Wazuh, Slack, Jira, VirusTotal, and more.

### 5. tracecat-workflow-patterns
Proven workflow patterns for alert triage, enrichment pipelines, compliance checks, and Splunk integrations.
